# Tarun Chandra - Portfolio Website

A modern, professional portfolio website built with Next.js showcasing AI/ML expertise, full-stack development skills, and freelance services.

## Features

- **Modern Design**: Clean, professional layout optimized for recruiters and clients
- **Responsive**: Mobile-first design that works on all devices
- **Performance Optimized**: Built with Next.js for fast loading and SEO
- **Interactive Elements**: Smooth animations and hover effects
- **Contact Form**: Functional contact form for project inquiries

## Sections

1. **Hero**: Dynamic role animation with key statistics
2. **About**: Value propositions and certifications
3. **Skills**: Technical expertise with progress bars
4. **Projects**: Featured projects with impact metrics
5. **Testimonials**: Client success stories
6. **Contact**: Service pricing and project inquiry form

## Technologies Used

- Next.js 14
- React 18
- CSS3 with CSS Variables
- Responsive Grid Layout
- Modern JavaScript (ES6+)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

Build for production:
```bash
npm run build
npm start
```

## Customization

- Update personal information in components
- Modify color scheme in `styles/globals.css` CSS variables
- Add/remove projects in `components/Projects.js`
- Update skills and percentages in `components/Skills.js`
